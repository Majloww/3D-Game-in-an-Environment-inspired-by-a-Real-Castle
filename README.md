# TVORBA 3D HRY V PROSTREDÍ INŠPIROVANOM SKUTOČNÝM HRADOM
## BAKALÁRSKA PRÁCA


Táto bakalárska práca sa zaoberá procesom tvorby 3D počítačovej hry, ktorej prostredie je inšpirované reálnym hradom. Cieľom práce je prepojiť historickú architektúru 
s technológiami herného vývoja a vytvoriť interaktívne prostredie. Teoretická časť práce opisuje základné pojmy súvisiace s počítačovými hrami, ich žánrové rozdelenie a charakteristiku najpoužívanejších herných enginov. Osobitná pozornosť je venovaná enginu Unity, jeho funkciám, štruktúre a používateľskému rozhraniu. Praktická časť sa zameriava na návrh, implementáciu a testovanie herného projektu vytvoreného v tomto engine. V rámci vývoja sa využíva jazyk C# a editor Visual Studio Code, pričom herný svet bol vytvorený podľa predlohy skutočného hradu s dôrazom na hrateľnosť a interakciu hráča. Výsledkom práce je funkčný 3D model historickej lokality spracovaný formou hry, ktorý demonštruje možnosti enginu Unity pri tvorbe edukatívnych a prezentačných aplikácií.

Kľúčové slová: 3D hra. Herný Engine. Unity. C#
